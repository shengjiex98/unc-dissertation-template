\section{How to use}

This document is a minimal starting point for a \gls{unc} thesis or
dissertation. Replace the metadata, front matter, and chapter text with
your own content. The official university formatting guide is a useful
reference when you need to verify submission requirements
\cite{unc-dissertation-guide}.

The bibliography setup in this template is intentionally small. It uses
BibLaTeX with a lightweight style derived from the mimosis template
\cite{mimosis-template}. Add your own entries to \texttt{thesis.bib}
and cite them in the usual way.
